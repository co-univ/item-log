# 진출자 선정

선착순 모드에서 풀이 진행 중 maker가 다음 라운드 진출자를 선정하는 별도 페이지. [풀이관리](./play-management.md) 1.6 참고.

## 화면 구성

3개 영역으로 나뉨:

| 영역 | 정렬 기준 | 비고 |
|---|---|---|
| **(1) 선착순 등수** | 가장 빠르게 많은 정답을 제출한 사람 순 | 공동 등수는 가나다순 |
| **(2) 정답자 등수** | 선착순 무관, 맞은 개수 기준 | 동일 등수는 쉼표로 묶어서 한 줄에 표시 |
| **(3) 다음단계 진출자** | (1)/(2)에서 선택한 사람 모음 | **직접 입력 가능** — `이하은(웃긴 감자)` 형식 정확히 |

## 선택 방식

- 등수 앞 체크박스 — 직접 선택.
- 하단 "선착순 N등까지" 입력 — N 입력 시 1~N등 자동 체크.

## 확정

`진출자 확정하기` → 모달 ("0000 진출자로 확정하시겠습니까?")
- **확인** → 풀이관리 페이지(1.1)로 복귀.
- **취소** → 진출자 선정 페이지(1.5)에 머무름.
