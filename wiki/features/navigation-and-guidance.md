# 내비게이션 / 사용자 안내

MAIT 안에서 사용자가 핵심 섹션으로 이동하고, 새 기능을 발견하고, 반복 방문 경로를 줄이는 흐름이다. 2026-05-04 회의에서는 새 기능 출시 안내와 메인 페이지 앵커링이 개선 후보로 올라왔고, 2026-05-10 회의에서 메인페이지 앵커값과 LNB/최근 방문 후속이 더 구체화됐다.

## 메인페이지 앵커링

- 메인페이지의 `MAIT 사용법` 섹션은 `#guide` 앵커를 사용한다.
- 메인페이지의 `주요 기능` 섹션은 `#features` 앵커를 사용한다.
- FE가 두 태그를 반영한다.
- 신규 `MAIT-142`로 추적한다.

근거: [2026-05-10 주간 회의록](../sources/2026-05-10-weekly-meeting.md).

## 새 기능 안내

- 새 기능은 업데이트 후 최초 진입 시 현재 온보딩과 같은 형식으로 간단히 안내하는 방향이다.
- 상세 설명이 필요하면 기능 설명 페이지를 만들고 링크로 연결한다.
- 릴리즈 노트는 별도로 쓰되, 인앱 안내는 다음 기능부터 적용을 고려한다.
- 이 흐름은 기존 `MAIT-140`과 연결된다.

근거: [2026-05-04 주간 회의록](../sources/2026-05-04-weekly-meeting.md), [2026-05-10 주간 회의록](../sources/2026-05-10-weekly-meeting.md).

## 온보딩 개선

- 온보딩은 기존 `MAIT-85`와 연결해 디자인 검토를 이어간다.
- 블러 처리 형태와 전/후 이동 버튼 UI가 개선 후보로 올라왔다.
- 실시간 랭킹 정보 조회 영역은 더 넓게 보여달라는 요청이 있었고, 프론트가 검토 후 공유하기로 했다.
- 적용 시점은 6월 말 후보로 언급됐지만 확정 일정은 아니다.

근거: [2026-05-14 주간 회의록](../sources/2026-05-14-weekly-meeting.md).

## 최근 방문 탭

- 사용자가 최근 접근한 공간이나 항목을 빠르게 다시 찾기 위한 `최근 방문 탭`이 프론트 후속으로 올라왔다.
- 아직 노출 위치, 최근성 기준, 빈 상태 처리, 개인 워크스페이스/LNB와의 관계는 정해지지 않았다.
- 신규 `MAIT-144`로 추적한다.

근거: [2026-05-10 주간 회의록](../sources/2026-05-10-weekly-meeting.md).

## 개인 워크스페이스 진입

- 팀이 없는 사용자는 로그인 시 개인 워크스페이스가 자동 생성되고 해당 워크스페이스로 이동한다.
- 로그인 전 `바로 시작하기` 버튼을 누르면 로그인 후 개인 워크스페이스 생성 및 이동으로 이어진다.
- 팀이 있는 사용자는 최근 방문 팀 기준으로 이동한다. player는 `문제 풀기`, maker는 `문제 관리` 탭으로 이동한다.
- 개인 워크스페이스는 maker에 해당하므로 항상 `문제 관리`로 이동한다.
- 최근 방문 팀 저장 작업은 후속으로 남아 있다.

근거: [개인 워크스페이스 명세서](../sources/2026-05-10-spec-personal-workspace.md).

## LNB 펼치기

- LNB를 쉽게 펼치는 UX 개선이 후속 작업으로 올라왔다.
- 클릭 영역, 펼침/접힘 상태 유지, 모바일/좁은 화면 대응 기준은 아직 정리되지 않았다.
- 신규 `MAIT-145`로 추적한다.

근거: [2026-05-10 주간 회의록](../sources/2026-05-10-weekly-meeting.md).

## 풀이 화면 이동 원칙과 연결

학습모드/복습모드 풀이 화면은 별도 나가기 버튼이나 브레드크럼을 두지 않고, LNB/NavBar 기반 이동으로 충분하다는 의견이 2026-05-10 회의에서 정리됐다. 이 결정은 [문제 풀기](./play-player.md)의 학습/복습 모드 동선과 연결된다.

## 미해결

- 새 기능 안내를 몇 회, 누구에게, 어떤 조건에서 다시 보여줄지.
- 온보딩 블러/전후 이동 버튼/실시간 랭킹 정보 영역의 최종 디자인과 적용 일정.
- 기능 설명 페이지의 위치와 작성 책임.
- 최근 방문 탭의 대상 단위와 저장/정렬 기준.
- 최근 방문 팀 저장 기준과 개인 워크스페이스 포함 여부.
- LNB 펼침 상태의 저장 여부와 모바일 동작.
