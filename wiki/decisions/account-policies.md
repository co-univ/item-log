# 계정 정책

회원가입·마이페이지에서 공통으로 적용되는 입력 정책 모음.

## 비밀번호

| 항목 | 값 |
|---|---|
| 최소 길이 | **8자 이상** |
| 필수 구성 | 영문 + 숫자 + 특수문자 (모두 포함) |
| UI 피드백 | 입력 필드 하단 조건 체크박스 (영문/숫자/특수문자) — 회색→초록 |
| 보기/숨기기 | 눈 아이콘 |
| 불충족 안내 | "조건을 충족하지 못했습니다. 다시 입력해주세요." |
| 변경 시 | 기존 비밀번호 입력 후 새 비밀번호 + 확인 일치 |

## 닉네임

| 항목 | 값 |
|---|---|
| 자동 생성 | `user1234` 패턴 (계정 생성 시) |
| 길이 | **2~20자** |
| 중복 | 불가 (중복확인 버튼) |
| 필터링 | 금지어 (비속어, 운영자 등 특정 단어) |
| 안내 — 기본 | "닉네임은 2~20자 이내로 입력할 수 있습니다." |
| 안내 — 중복 | "이미 사용 중인 닉네임입니다." |
| 안내 — 사용 가능 | "사용 가능한 닉네임입니다." |

## 이름

| 항목 | 값 |
|---|---|
| 허용 문자 | 한글 / 영문만 (특수문자·숫자 불가) |
| 길이 | 2~20자 |
| 수정 | **불가** (가입 시 입력값 고정) |
| 안내 | "실명을 입력해주세요. (특수문자와 숫자는 입력할 수 없습니다.)" |

## 전화번호

| 항목 | 값 |
|---|---|
| 형식 | 숫자만 (예: `01012345678`) |
| 중복 | 불가 — 한 번호로 여러 계정 가입 불가 |
| 안내 — 기본 | "숫자만 입력해주세요. (예: 01012345678)" |
| 안내 — 중복 | "이미 등록된 전화번호입니다." |

## 이메일

| 항목 | 값 |
|---|---|
| 형식 검증 | `@` 포함, 도메인 포함, 공백 불가 |
| 중복 | 불가 (이미 가입된 이메일 차단) |
| 인증 유효시간 | **10분** (인증메일 발송 후) |
| 수정 | **불가** |

## 약관

- 서비스 이용약관 (필수)
- 개인정보 수집 및 이용 (필수)

모두 체크해야 가입하기 활성화.
