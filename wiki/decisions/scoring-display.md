# 채점 / 정답 표시 정책

복습 모드에서 답 제출 후 색상 표시 방식. 실시간/학습 모드는 현 동작 유지.

## 모드별 정책

| 모드 | 채점 표시 정책 |
|---|---|
| 실시간 | 하나라도 틀리면 오답 (현 상태 유지) |
| 학습 | 하나라도 틀리면 오답 (현 상태 유지) |
| 복습 | **답 개수에 따라 다름 — 아래 표 참고** |

## 복습 모드 — 단답 vs 다답

### 단답 (객관식 1개 / 주관식 1개 / 빈칸 1개 / 순서)

- 변동 없음. 전체가 하나의 정답 단위.
- 맞으면 **초록**, 틀리면 **빨강**.

### 다답 (객관식 2개 이상 / 주관식 2개 이상 / 빈칸 2개 이상)

- **부분 채점**: 사용자가 선택·작성한 답 단위로 판정.
- 맞은 답은 **초록**, 틀린 답은 **빨강**.

## 채점 노출 시점

- 학습 모드: 제출 후 채점 뷰에서 **정/오답 + 풀이/해설 한꺼번에**.
- 복습 모드: 제출 후 **채점만 표시**, 해설은 해설보기 클릭 시 노출.
- 실시간 모드: 제출 후 정답 / 오답 여부만 즉시 노출.
