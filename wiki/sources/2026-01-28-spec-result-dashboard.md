# 풀이결과 대시보드 명세서

풀이가 끝난 뒤 player가 결과를 확인하는 대시보드. **메인**(전체 랭킹·개인 통계·문제셋 목록) + **상세**(문제셋별 랭킹·오답률 top3·점수·문제별 결과) 2단 구조.

## 핵심 요약

### 메인 대시보드
- **우리팀 랭킹** (전체) — 실시간 모드 집계만. 선착순/정답수 기준 탭 전환. 1~3등(또는 5등) + 내 등수.
- **개인 정답률 원그래프** — 푼 문제셋 개수, 맞은 문제 수, 개인 정답률.
- **카테고리별 정답률 바그래프** — 카테고리(=문제셋 주제 태그)별. 상위 8개 + "더보기" 토글.
- **문제셋 정보 카드** — 상세 대시보드 진입점. 제목/날짜/우승자/내 정답률 vs 평균/자세히보기.

### 상세 대시보드
- **진입**: 메인 1.4 카드 또는 풀이 종료 후 **우승자 화면의 "풀이결과 보러가기"**.
- **우리팀 랭킹** (해당 셋) — 실시간이면 선착순, 학습이면 정답수.
- **오답률 top3** — 1~3위 질문 + 오답률 + 문제 확인하기.
- **점수** — `00점 (00/00)`. **탈락해도 전체 문제 수 기준** (예: 10문제 중 5번에서 탈락 → 40점).
- **전체 문제 풀이결과** — 복습풀이+해설보기 상태와 동일. 다시 풀기 불가, 확인만. 안 푼 문제는 오답 처리되고 정답·풀이 노출.
