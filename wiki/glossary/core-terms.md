# 핵심 용어

mait 명세 전반에서 반복 사용되는 용어. 새 source가 추가되면 여기 갱신.

## 권한 / 역할

- **owner** — 팀 소유자(팀당 1명). 멤버 리스트에서 삭제 불가, maker 영역 하단에 별도 표시. 권한 양도 흐름 미정.
- **maker** — 문제 편집 권한자. 문제셋 생성·편집·풀이관리·카테고리 관리·멤버 초대(권한 따라).
- **player** — 문제 풀이자. 풀이만 가능.

상세는 [decisions/permission-roles.md](../decisions/permission-roles.md).

## 콘텐츠 단위

- **문제셋 (problem set)** — 1회 풀이의 단위. 여러 질문(question)을 묶음.
- **질문 (question)** — 문제셋 안의 단일 문항. "문제"로도 표기되나 워딩 통일은 "질문"으로 진행 중.
- **선지** — 객관식의 보기. 최소 2개, 최대 8개.
- **답안** — 주관식·빈칸·순서의 정답.
- **인정답안** — 표기 차이가 있어도 정답으로 인정하는 변형. 답안마다 최대 5개. 정답 영역에는 노출 안 됨.
- **보기** (순서 유형) — player에게 보여주는 항목. 알파벳 자동 부여, 변경 불가.
- **카테고리** — 문제셋 분류 태그. 다중 선택, 무제한 추가, [카테고리 관리](../features/category-management.md)와 연동.

## 풀이 방식 / 모드

- **실시간 풀이** (선착순) — player가 동시 참여. 진행 예정 → 진행 중 → 종료됨.
- **학습 풀이** — 세션 없이 개별 학습. 모든 인원 풀이 완료 시 자동 종료.
- **복습** — 실시간/학습 풀이 종료 후 재방문하는 상태. 공개 범위 설정 가능.

## maker 측 풀이 제어

- **문제공개 토글** — player에게 문제 화면 노출 ON/OFF. OFF면 대기화면.
- **제출허용 토글** — player의 제출 버튼 활성화 ON/OFF. ON 시 종 애니메이션. 문제공개 ON이어야 사용 가능.
- **득점자** — 가장 빠르게 정답 제출한 사람. 인정답안 추가로 정답이 바뀌면 자동 갱신.
- **진출자** — 다음 라운드로 보낼 player. 선착순/정답자 등수 + 직접 선택/입력.

## player 측 상태 (학습 모드)

- **풀이 전** — 시작은 됐지만 풀지 않음.
- **풀이 중** — 풀고 있고 자동저장 동작 중.
- **풀이 완료** — 채점하기 누른 후. "통계확인" 버튼으로 변경.

## 저장

- **임시저장** — 수동, 명시적 마커. 첫 임시저장 시 제목·주제 입력 강제.
- **자동저장** — 1분 단위 (생성 단계) / 다른 문제로 이동 시 (학습 모드 풀이).
- **임시 저장본n** — 자동저장만 된 채 화면 이탈 시 메인에 표시되는 자동 명명.

## 공개 범위 (복습 상태에서만)

- **전체 공개** / **그룹 공개** / **비공개** — [decisions/visibility-states.md](../decisions/visibility-states.md).

## 검색 / 북마크

- **북마크** — 컬렉션에 문제셋을 저장·열람. **팀으로 가져오거나 편집은 불가**.
- **컬렉션** — 북마크의 저장 단위. 추가 가능. 자체 정의는 별도 명세 필요.
- **개인 학습 공간** — "내 그룹에 추가" 시 선택 가능한 default 공간. 정의는 명세 누락.

## 결과 / 통계

- **우승자 화면** — 풀이 종료 직후 player에게 노출되는 화면. "풀이결과 보러가기" 진입점 포함. 자체 명세는 누락.
- **풀이결과 대시보드** — [features/result-dashboard.md](../features/result-dashboard.md). 메인(전체 통계) + 상세(셋별 결과).
- **우리팀 랭킹** — 메인은 실시간 모드 전체 집계, 상세는 해당 셋 집계.
- **개인 정답률** — 푼 문제셋 개수 대비 맞은 문제 수 비율. 원그래프.
- **카테고리별 정답률** — 카테고리(=문제셋 주제 태그)별 정답률 바그래프. 상위 8개 + 더보기.
- **평균 정답률** — 문제셋 카드에서 "내 정답률 vs 전체 평균" 비교용. 평균 단위(팀/서비스) 미정.
- **오답률 top3** — 상세 대시보드에서 오답률 높은 질문 1~3위.
- **점수** — `00점 (00/00)` 형식. **분모는 항상 전체 문제 수** — 탈락해도 분모 유지. [decisions/score-calculation.md](../decisions/score-calculation.md).

## 그룹 / 팀

- **그룹/팀** — 문제셋 단위로 maker/player를 묶는 공간. 명세에서 그룹과 팀이 혼용됨 (정확한 구분 명세 누락).

## 모드 토글 (3.7 주관식)

- **정답 개수 조절 모드** — maker 답안 개수 ≠ player 요구 답안 개수일 때 사용하는 토글.

## 계정 입력 항목

- **이메일** — 로그인 ID. 인증 메일 10분 유효. 수정 불가.
- **이름** — 한글/영문 2~20자. 수정 불가.
- **닉네임** — `user1234` 자동 생성, 2~20자, 금지어 필터, 중복 불가. 수정 가능.
- **전화번호** — 숫자만. 한 번호로 다중 계정 불가.

상세는 [decisions/account-policies.md](../decisions/account-policies.md).
