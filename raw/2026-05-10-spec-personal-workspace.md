# 개인 워크스페이스

| **구분** | **세부기능** | **상세설명** | **의견** |
| --- | --- | --- | --- |
| 메인 페이지 플로우 | 로그인 및 초기 진입 | 팀이 없는 경우: 로그인 시 개인 워크스페이스 자동 생성 후 해당 워크스페이스로 이동 로그인 전: ‘바로 시작하기’ 버튼 클릭 → 로그인 → 개인 워크스페이스 생성 및 이동 |  |
|  | 최근 방문 팀 이동 | 팀이 있는 경우 최근 방문 팀 기준으로 이동 player: ‘문제 풀기’ 탭 이동 maker: ‘문제 관리’ 탭 이동 (개인워크스페이스는 maker에 해당하여 항상 문제 관리로 이동) | 최근 방문 팀 저장 작업 필요 |
| 문제 관리 | 상태처리 및 UI | 본인이 모든 문제 풀이 완료 시 완료 상태로 변경 미트볼 및 버튼 UI 기존과 동일 유지 |  |
|  | 제외기능 | 실시간 모드 제외 풀이 시작 및 종료 기능 제외 |  |
| 문제 풀기 | 모드 | 학습 모드, 복습 모드만 제공 실시간 풀이 기능 제외 |  |
| 풀이 결과 대시보드 | UI | 랭킹 영역 제외 정답률 영역 상단 배치 |  |
| 팀관리 | 카테고리 관리 | 팀관리 → 카테고리 관리로 명칭 수정 카테고리 관리 화면은 기존과 동일하게 유지 |  |
|  | 제외기능 | 멤버관리 비활성화 |  |
