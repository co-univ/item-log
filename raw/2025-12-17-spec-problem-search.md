| **(비회원) 문제 검색** | 1.1. 검색바 | 검색 가능한 내용: 제목/팀명/문제셋 질문 | ~~검색 안 한 상태의 화면 → 완료~~ |
|  | 1.2. 검색한 뒤 문제셋 리스트 | 리스트에서 노출하는 정보 노출 가능한 조건: 전체공개 && 복습모드  제목, 팀명, 문제수, (해당 검색어가 질문에 있다면) 질문, (해당검색어가 질문에 없다면) 1번부터 질문, 북마크 수 | ~~북마크 수는 북마크 옆에 표시되도록~~ 카드 → 리스트 형태로 변경 |
|  | 1.3. 문제셋 상세  | 문제셋 제목, 팀명, 날짜 노출 문제 2문제 노출 (풀이는 불가, 확인만 가능) 그 이후 문제는 노출 x → 로그인 화면으로 유도 |  |
| (회원) 문제 검색 | 2.1. 검색바  | 검색 가능한 내용: 제목/팀명/문제셋 질문 |  |
|  | 2.2. 검색한 뒤 문제셋 리스트 | 리스트에서 노출하는 정보 노출 가능한 조건: 전체공개 && 복습모드  제목, 팀명, 문제수, (해당 검색어가 질문에 있다면) 질문, (해당검색어가 질문에 없다면) 1번부터 질문, 북마크 수 북마크: 컬렉션에 문제셋을 저장하고 열람할 수 있지만, 팀(개인 학습공간 포함)으로 문제를 가져오거나 편집하는 것은 불가능한 기능  북마크 아이콘 선택 시, 북마크할 컬렉션 선택 컬렉션 추가 가능 저장하기 → 해당 문제셋이 컬렉션에 저장되며 북마크 아이콘 채워짐 | 카드 → 리스트 형태로 변경 |
|  | 2.3. 문제셋 상세  | 각 문제 체크박스: 문제 별로 체크박스 선택 가능  전체 선택: 모든 문제가 선택되며, 모든 문제의 체크박스가 선택되어 표시됨 내 그룹에 추가 체크박스가 하나라도 선택되어야 활성화 상태 클릭 시, 추가할 수 있는 팀 노출  개인 학습 공간, maker(owner포함)로 속해있는 팀 중에 선택 가능 다중 선택 가능  북마크: 컬렉션에 문제셋을 저장하고 열람할 수 있지만, 팀(개인 학습공간 포함)으로 문제를 가져오거나 편집하는 것은 불가능한 기능  북마크 아이콘 선택 시, 북마크할 컬렉션 선택 컬렉션 추가 가능 저장하기 → 해당 문제셋이 컬렉션에 저장되며 북마크 아이콘 채워짐 |  |
