## 원문 스냅샷

### 공지사항

- `출시가 2주도 안남았다`는 문장으로 전체 긴박도를 공유했다.

### 백엔드/프론트 블로커

- 백엔드는 `문제 생성`, `풀이 관리`, `학습 모드? 복습 모드?에 대한 준비`, `메인 서버 배포`, `로그 서버 세팅`, `Lexo Rank 재배치 로직`, `정책 API 작업`, `문제 셋 검색 기능`, `팀 초대 승락`을 남겼다.
- `정책 관련 코드리뷰 - 반영할게 많`과 정책 버전 SQL 메모가 같이 기록됐다.
- 프론트는 `문제 셋 생성하기 페이지 하나 남음`, `풀이 관리`, `소셜로그인 연동`을 공유했다.

### 운영 메모

- 프론트 역할 분배와 마이페이지 정리 필요성이 추가 메모로 남았다.

## AI 후속 블록 스냅샷

### 6. AI의 요약

- 11월 첫째 주 회의는 출시 직전 남아 있는 백엔드/프론트 블로커를 한 번에 펼쳐 놓고 우선순위를 다시 보는 회의였다.
- 검색, 팀 초대 승락, 풀이관리, 학습/복습 준비가 모두 같은 레벨의 `출시 전 처리해야 할 일`로 올라왔고, 정책 API와 로그/배포처럼 운영 레벨 작업도 동시에 등장했다.
- 특히 `정책 관련 코드리뷰`, `정책 API 작업`은 권한/정책 논의가 문서 수준을 넘어 실제 API 반영 단계로 이동했음을 보여준다.
- 프론트 쪽은 생성하기 마지막 페이지와 풀이관리, 소셜 로그인 연동을 끝내야 했고, 마이페이지 역할 분배까지 겹치면서 일정 압박이 매우 큰 상태였던 것으로 읽힌다.

### 7. 할일 티켓 정리

- (신규) 팀 초대 승락 및 초대 세부사항 전달 흐름 구현 - MAIT-130
- (신규) API 권한 관리 정책 정리 및 반영 - MAIT-132
