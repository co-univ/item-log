# 로그인/회원가입/마이페이지

| **구분** | **세부 기능** | **상세 설명** |
| --- | --- | --- |
| 로그인 | 1.1 소셜 로그인 구글/카카오/네이버 | 인증 성공 시 사용자 이름/이메일 수신 실패(취소/네트워크/권한거부) 시 표준 에러 토스트 및 재시도 안내 |
|  | 1.2 로컬 로그인 | `이메일로 로그인` 버튼 클릭 시 이메일 로그인 화면으로 이동 이메일 주소/비밀번호 입력 [눈 아이콘]으로 보기/숨기기 가능 로그인 유지  이메일/비밀번호 입력값 발생 시 `로그인` 버튼 활성화 로그인 실패 시 안내문구 잘못된 이메일/비밀번호 → "아이디 또는 비밀번호가 올바르지 않습니다." |
| 회원가입 | 2.1 소셜 로그인 최초 진입 후 모달 | 모달명: 계정 설정하기  약관 동의(필수/선택) 닉네임 입력(자동 랜덤 제안, `중복확인` 버튼 클릭 시 중복검사) 안내분구 기본: "닉네임은 2~20자 이내로 입력할 수 있습니다." 중복 시: "이미 사용 중인 닉네임입니다." 사용 가능 시: "사용 가능한 닉네임입니다." 필수 동의 체크+닉네임 유효 시 `시작하기` 버튼 활성화 로그인 완료 후 메인 화면 우측 최상단 `로그인 → 이름`으로 변경 `닫기` 버튼 클릭 시 로그인 실패 |
|  | 2.2 로컬 회원가입 | **이메일(아이디) 입력 및 인증** 사용자는 이메일 주소를 입력 후 [이메일 인증] 버튼 클릭 형식 검증: `@` 포함, 도메인 포함 여부, 공백 불가 중복 가입 여부 체크 (이미 가입된 이메일인지) 인증 메일 발송 → 메일 내 "인증하기" 버튼 클릭 시 인증 완료 처리 (이메일 전송 후 10분까지) 안내문구 기본: "이메일 주소를 입력해주세요." 형식 오류 시: "올바른 이메일 주소 형식이 아닙니다." 인증 요청 전: "입력한 이메일로 인증 메일을 발송합니다." 인증 완료 후: "이메일 인증이 확인되었습니다." **비밀번호 입력** 조건: 최소 8자 이상, **영문 + 숫자 + 특수문자** 포함 필수 입력 필드 하단에 조건 체크박스(`영문`, `숫자`, `특수문자`)를 실시간 표시 (회색 → 초록색) [눈 아이콘]으로 보기/숨기기 가능 조건 불충족 시 안내문구 "조건을 충족하지 못했습니다. 다시 입력해주세요." **비밀번호 확인** 첫 번째 비밀번호와 일치 여부를 검증 실시간 비교 후 불일치 시 에러 표시 안내문구 기본: "비밀번호를 다시 입력해주세요." 불일치 시: "입력한 비밀번호가 일치하지 않습니다." **이름** 필수 입력 값 한글/영문만 허용, 특수문자 및 숫자 입력 불가 2~20자 길이 제한 기본 안내문구 실명을 입력해주세요. (특수문자와 숫자는 입력할 수 없습니다.) **전화번호** 숫자만 입력 가능 (예: `01012345678`) 입력값에 대해 `중복확인` 버튼 클릭 시 중복검사 (한 번호로 여러 계정 가입 불가) 안내문구 기본: "숫자만 입력해주세요. (예: 01012345678)" 중복 시: "이미 등록된 전화번호입니다." **닉네임** 계정 생성 시 자동 생성(예: `user1234`)되며 사용자가 직접 수정 가능 입력값에 대해 `중복확인` 버튼 클릭 시 중복검사 길이 제한: 2~20자 금지어(비속어, 운영자 등 특정 단어) 필터링 필요 안내문구  기본: "닉네임은 2~20자 이내로 입력할 수 있습니다." 중복 시: "이미 사용 중인 닉네임입니다." 사용 가능 시: "사용 가능한 닉네임입니다." **약관 동의** (필수) 서비스 이용약관 동의 (필수) 개인정보 수집 및 이용 동의 모든 필수 항목이 체크되어야 최종 가입 버튼 활성화 **가입하기 버튼** 모든 필드 유효성 검증 + 이메일 인증 + 중복 확인 + 필수 약관 동의 완료 시 활성화 제출 시 로딩 표시 및 중복 클릭 방지 성공 시 계정 생성 완료 및 로그인 → 서비스 초기 화면으로 이동 |
| 마이페이지 | 3.1 개인 정보 확인 | 사용자 계정의 기본 정보(이메일, 이름, 전화번호, 닉네임, 비밀번호)를 화면에 표시 이메일과 이름은 가입 시 입력된 값을 보여주며 수정 불가 전화번호, 닉네임, 비밀번호는 현재 등록된 값을 확인 가능 |
|  | 3.2 개인 정보 수정 | **전화번호 수정**: 숫자만 입력 가능, [중복체크] 버튼 제공, 이미 등록된 번호일 경우 오류 메시지 표시 **닉네임 수정**: 자동 생성된 닉네임을 사용자가 변경 가능, 입력 시 [중복체크] 또는 실시간 검증 수행, 2~20자 제한 및 금지어 필터링 적용 **비밀번호 변경**: 기존 비밀번호 입력 후 새 비밀번호와 확인 비밀번호를 입력, 조건(영문+숫자+특수문자 포함 8자 이상)을 충족해야 하며 두 입력값이 일치해야 수정 가능 수정 완료 시 성공/실패 메시지 노출 및 변경 내역 반영 |
