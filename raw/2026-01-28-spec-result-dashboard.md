# 풀이결과 대시보드

| **구분** | **세부기능** | **상세설명** | **코멘트** |
| --- | --- | --- | --- |
| 풀이결과 대시보드 메인 | 1.1. 우리팀 랭킹 (전체랭킹)  | 선착순 기준, 정답수 기준 탭 전환 전체 문제셋 기준 랭킹 실시간 모드의 순위만을 집계 1,2,3등까지 노출 (디자인에 따라 5등까지로 변경 가능) 이름 + 닉네임 노출 내 등수노출 (몇등인지와 상관없이) |  |
|  | 1.2. 정답률 그래프 (개인 통계) | 노출하는 정보 지금까지 푼 문제셋 개수 그 중 맞은 문제 수  위 정보를 계산한 개인 정답률 → 원그래프로 표현 |  |
|  | 1.3. 카테고리별 정답률 바 | 주제별 카테고리를 기반으로 각 주제별 정답률을 바그래프로 표현  카테고리 = 문제셋의 주제 태그   카테고리는 상위 8개만 노출 이후 카테고리는 더보기를 버튼을 통해 확인 가능 (토글 형식) |  |
|  | 1.4. 문제셋 정보 | 문제셋 별 상세 대시보드인 2로 이동하는 진입점의 역할을 함 노출하는 정보 문제셋 제목 풀이날짜 우승자 이름 내 정답률/전체 평균 정답률 자세히보기 버튼  |  |
| 풀이결과 대시보드 상세 진입점 1.4.에서 진입 문제 풀기 끝난 뒤 우승자 화면의 '풀이결과 보러가기' | 2.1. 우리팀 랭킹 (해당 문제셋 랭킹) | 해당 문제셋 기준 랭킹 실시간 모드인 경우 - 선착순 기준  학습 모드인 경우 - 정답수 기준  |  |
|  | 2.2. 오답률 top3 | 오답률 높은 문제 1위부터 3위까지 문제 질문, 오답률, '문제 확인하기' |  |
|  | 2.3. 점수 | 형식: 00점 (00/00) 해당 문제셋의 내 점수 노출: 중간에 탈락해서 뒷 문제를 못풀었어도 전체 문제개수 기준 점수로 계산 eg. 10문제 중 5번문제에서 탈락하여도 40점(4/10)으로 표시 |  |
|  | 2.4. 전체 문제 풀이결과 | 전체 문제의 풀이결과 노출: 복습풀이+해설보기를 한 상태와 동일 상단 번호에 정답 유무에 대한 표시 필요(디자인) 2.2의 문제 확인하기 클릭 시 해당 문제로 이동 문제를 다시 풀 수 있는 영역은 아니며, 단순히 문제와 내가 고른 정답, 실제 정답, 풀이를 확인할 수 있는 곳 (중간에 탈락하여) 풀지 않은 문제의 경우 오답표시되며, 정답 및 풀이 노출됨 |  |
