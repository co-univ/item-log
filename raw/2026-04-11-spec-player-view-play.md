## 추가 메모

### 복습 (자유 노트)
- 복습문제는 여러번 풀 수 있음. 한번만 풀고 끝내기에는 복습이라는 의미와 어긋남.
  - 다시 풀 수 있게끔 하기 위한 처리 필요.
  - 이탈 후 화면
- 복습문제는 이미 종료 → 복습모드로 이동한거라서 문제 관리 페이지에서 문제 수정, 풀이관리 버튼이 필요 없음
  - 대신 추가할 내용 고려.

### 12/10 기획논의

**복습모드**

관리뷰
- 문제 생성 시 학습모드나 실시간모드는 팀 종속적인 문제셋을 제작하는 것이므로 그룹공개가 디폴트값 (문제 생성 시 공개대상 정하는 플로우 삭제)
  - 근거: 팀에서 문제 풀이 중에 제3자가 문제에 참여하면 안 됨. 문제 풀이 시에는 문제정보가 계속적으로 변경될 수 있기 때문
- 종료됨 상태 > 복습모드로 이동 시에만 공개대상을 정함. 따라서 문제셋 검색은 복습모드의 문제셋만 검색 가능함
- 문제수정, 풀이관리 버튼 북마크 수 표시

풀이뷰
- 문제 번호 한번 이동하면, 이전 문제는 무조건 정답 off 풀이 off
- 문제셋 이탈했다가 다시 돌아와도 무조건 정답 off 풀이 off

---

**생성하기 페이지 + 문제정보입력 페이지 및 플로우 수정 필요**

- 공개 대상 선택이 필요 없음 (복습모드 설명 참고)
- 제목 입력란을 문제 생성 페이지 전체에서 보이도록 수정 예정임
- 문제 관리의 카드에서 문제 제목 수정 바로 가능
- 문제 번호의 가장 앞쪽에 문제 정보를 담고있는 아이콘 추가 → 클릭하면 문제 정보 확인 가능: 기존 문제정보에 제목, 풀이방식 추가
- 생성하기 클릭 시 → 바로 문제 생성

### 검색 (자유 노트)
- 카드 → 리스트형태로 변경
- 검색가능한 내용: 제목, 팀명, 문제셋 질문
- 리스트에서 노출하는 정보: 제목, 팀명, 문제수, (해당 검색어가 질문에 있다면) 질문, (해당검색어가 질문에 없다면) 1번부터 질문, 북마크 수
