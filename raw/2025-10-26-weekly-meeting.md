# 2025-10-26 주간 회의록

## 메타

- Notion 페이지: <https://www.notion.so/29487d592b6e80c496feed97e408c448>
- 상태: 완료
- 구분: 주간
- 비고: 2026-04-25에 AI 요약과 티켓 정리 블록을 추가했다.

## 원문 스냅샷

### 핵심 논의

- AI 생성 요청 시 대기 시간과 사용자 안내 방식
- 공통으로 쓸 로딩중 화면과 에러 페이지 필요성
- 팀관리 기획 초안 공유

### 세부 메모

- 업로드와 AI 처리 사이 체감 대기 시간이 존재
- 기능별 임시 화면이 아니라 공통 상태 화면이 필요
- 팀관리 초안에는 초대 링크 생성/관리, 계정 제거, 권한 변경, 승인/거절 흐름이 포함

## AI 후속 블록 스냅샷

### 6. AI의 요약

- AI 문제 생성 요청 이후의 체감 대기 시간, 공통 로딩/에러 상태, 팀관리 기획 초안이 중심이 된 회의였다.
- AI 생성은 업로드와 생성 처리 사이에 기다림이 길어 사용자가 현재 상태를 이해할 수 있는 화면이 필요했고, 기능별 임시 대응보다 공통 상태 패턴을 먼저 정리해야 한다는 맥락이 드러났다.
- 팀관리 초안은 초대 링크 생성/관리, 계정 제거, maker/player 권한 변경, 승인/거절 흐름까지 넓게 포함했고, 다음 회의에서 세부 정책을 더 다듬는 전제의 초안 단계였다.

### 7. 할일 티켓 정리

- 기존 티켓 재사용 1건, 신규 생성된 티켓 1건
- (기존) AI 문제 생성 API 연동 및 운영 안정화 - MAIT-131
- (신규) 공통 로딩/에러 상태 화면 정리 - MAIT-134
